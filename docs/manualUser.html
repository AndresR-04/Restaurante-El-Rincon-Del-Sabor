ALCANSE

Este proyecto consistirá en desarrollar un aplicativo web basados en el punto de venta y modulo post del restaurante, que permitirá llevar una mejor organización sobre el registro de ventas que se hacen, así el administrador y empleado en especifico podrán tener registros de la venta diaria con sus respectivas fechas y pedidos hechos por los clientes. También incluirá:

  • El software será implementado en la empresa El Rincón Del Sabor
  • El software utilizado por la secretaria y el gerente
  • El software implementa una interfaz grafica el cual este contendrá el login para poder acceder de manera correcta al software, también incluirá dentro del software botones, estos registrarán la información de la empresa, esa información será guardada en bases de datos que, desde ella se podrán hacer inventarios, informes y lo mas importante guardar la información de forma segura de la empresa.

    • REQUISITOS DE CONOCIMIENTOS PARA MANEJAR EL SISTEMA

    Para el uso correcto del aplicativo, el usuario deberá tener en cuenta los siguientes conocimientos generales informáticos para una mejor experiencia en su uso. Estos son:

  • Uso de navegadores como Chrome o Firefox

  • conocimientos de herramientas informáticas

  • Manejo básico del sistema operativo Windows

  • Conocer las políticas de seguridad del aplicativo sugeridas por la empresa, por ejemplo, ingreso de usuario, validar información, manejos de contraseñas, etc.


    • REQUISITOS DE HARDWARE

  • Computador con un mínimo de 4 GB de RAM
  • Espacio libre en el disco duro de 500 MB
  • Conexión a internet con mínimo de 10 MB

    • REQUISITOS DE SOFTWARE

  • Sistema operativo Windows 10, Linux o MacOs 10
  • Lector de pdf Acrobat Reader o similar

    • POLÍTICAS DE SEGURIDAD

    La empresa GARVSOFT cuenta con una política de seguridad que debe cumplirse de manera correcta para no cometer infracciones o faltas al momento de usar el aplicativo, tales como:

    Empleado

    1. Uso correcto del aplicativo web al realizar ventas
    2. Manejo correcto del roll que se le asigna a cada empleado
    3. Colocar contraseñas fuertes al crearse la cuenta
    4. No compartir cuentas
    5. No compartir contraseñas
    6. No compartir datos de algún cliente con otro empleado

    Gerente

    1. No divulgar la información personal de los clientes guardados en el aplicativo

    INFORMACIÓN SOBRE LA MESA DE AYUDA

    Los usuarios líderes que formarán la mesa de ayuda seremos los desarrolladores, ya que cada uno está en las condiciones para dar solución a algún problema que se presente con el aplicativo, dándole el usuario uso al mismo. Es por ello, que se deberá ir a la mesa de ayuda siempre y cuando un proceso dentro de la app, sea complicado o poco intuitivo, esos casos serán:

    1. Crear los platos dentro una categoría (el usuario se pierda en su ruta)
    2. Creación de una categoría (al llenar los datos, ingrese alguno que el sistema no esté permitido para ese modulo)
    3. Creación de un cliente (al llenar los datos, ingrese alguno que el sistema no esté permitido para ese modulo)
